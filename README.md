# Sneakers Store - Магазин кроссовок

## Реализованный функционал

- **Каталог кроссовок** с загрузкой данных из REST API
- **Сортировка** по названию и цене (дешевые / дорогие)
- **Поиск по названию** с фильтрацией по введенной строке
- **Корзина** в виде выезжающего справа дровера:
  - Добавление/удаление товаров
  - Подсчет итоговой суммы
  - **Автоматический расчет налога 5%**
- **Оформление заказа**: отправка содержимого корзины на сервер и очистка корзины после успешного запроса
- **Избранное**:
  - Добавление/удаление товаров в избранное
  - Синхронизация состояния избранного с сервером
- **Сохранение корзины в `localStorage`**:
  - Восстановление содержимого корзины при перезагрузке страницы
  - Реактивное обновление сумм и значков "товар в корзине"
- **Состояния пустых списков**:
  - Пустая корзина
  - Пустой список избранного
  - Отдельные информационные блоки с описанием действий для пользователя
- **Маршрутизация** между страницами:
  - Главная (`/`) — список всех кроссовок
  - Избранное (`/favorites`) — список избранных товаров
- **Переиспользуемые компоненты** (`Card`, `CartList`, `Drawer`, `InfoBlock`, `TheHeader`), разделенные по ответственности
- **UI на utility‑классах** (Tailwind‑подобная верстка) с плавными hover‑эффектами на карточках и кнопках
